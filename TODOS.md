### 드래그 대체 UI 상세 설계
- **What:** 카드 길게 누르기 → 사분면 선택 컨텍스트 메뉴의 구체적 UI 디자인
- **Why:** 접근성 필요 사용자 + 작은 화면에서 드래그가 어려운 경우 대체 수단 필요
- **Pros:** 접근성 향상, 터치 친화적 대안 제공
- **Cons:** 추가 UI 컴포넌트 구현 필요
- **Context:** design-brief.md 접근성 섹션에 "드래그 불가 환경에서 카드 길게 누르기 → 사분면 선택 메뉴 표시"로 언급했으나, 메뉴 디자인 (위치, 스타일, 애니메이션)은 미정의
- **Depends on:** 매트릭스 뷰 기본 구현 완료 후
