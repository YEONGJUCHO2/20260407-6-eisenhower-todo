# ⚔️ 아이젠하워 투두 — 긴급한 것은 중요하지 않다

## 씽크스미스 EP06 웹앱 기획서 v1

---

## 1. 개요

- **앱 이름:** (미정, 후보: 아이젠하워 투두 / 매트릭스 플래너 / 포커스 매트릭스)
- **위인:** 드와이트 D. 아이젠하워
- **방법론:** 아이젠하워 매트릭스 (긴급/중요 4사분면)
- **한 줄 정의:** 할 일을 긴급/중요로 분류하고, 내가 시간을 어디에 쓰는지 보여주는 투두 앱
- **핵심 명언:** "긴급한 것은 중요하지 않고, 중요한 것은 긴급하지 않다."
- **타겟:** 20~30대 직장인/대학생 (바이럴형)
- **디자인 톤:** (미정, 추후 논의)

### 1-1. 킥 (차별화 포인트)

**킥 1: 아이젠하워 매트릭스 시각화 + 시간 사용 유형**
- 할 일을 4사분면에 배치하고, 주간 데이터로 "나는 어떤 유형인가" 분석
- 소방관형, 전략가형 등 유형 부여 → 공유 카드 → 바이럴

**킥 2: 마크다운 파일 기반 (옵시디언 연동)**
- 할 일 데이터가 .md 파일로 내보내기/가져오기 가능
- 옵시디언 볼트에 바로 넣으면 노트로 활용 가능
- "내 데이터는 내가 소유"

### 1-2. 핵심 인사이트

대부분의 사람들이 1사분면(긴급+중요)에만 시간을 쓰고, 2사분면(중요+안긴급)은 무시한다. 근데 진짜 인생을 바꾸는 건 2사분면이다. 운동, 공부, 관계, 자기개발 — 다 긴급하진 않지만 중요한 것들.

## 2. 필수 기능 (과제 요구사항)

- ✅ 캘린더 — 날짜별 할 일 관리
- ✅ 할 일 추가 — 텍스트 입력 + 긴급/중요 선택 + 반복 주기 설정
- ✅ 할 일 완료/삭제 — 체크 또는 삭제
- ✅ 할 일 목록 표시 — 캘린더 뷰 + 매트릭스 뷰
- ✅ 킥 구현 — 아이젠하워 매트릭스 + 시간 사용 유형 + 마크다운 연동
- ✅ 실제 배포 — Vercel

## 3. 사용자 플로우

```
[캘린더 뷰] ←탭 전환→ [매트릭스 뷰]

캘린더 뷰: 날짜별 할 일 목록 (언제 하느냐)
매트릭스 뷰: 같은 할 일을 4사분면으로 (뭐가 중요하냐)
```

### 3-1. 할 일 추가 플로우

```
[할 일 입력]
"EP06 기획서 작성"
    ↓
[사분면 선택]
○ 🔴 즉시 실행 (긴급 + 중요)
● 🔵 계획 수립 (중요 + 안긴급)
○ 🟡 위임/최소화 (긴급 + 안중요)
○ ⚫ 제거 (안긴급 + 안중요)
    ↓
[반복 설정]
○ 없음 (일회성)
○ 매일
○ 매주 [요일 선택]
○ 매월 [날짜 선택]
○ 매년 [월/일 선택]
    ↓
[추가 완료]
→ 캘린더에 표시 + 매트릭스에 배치
```

### 3-2. 전체 플로우

```
[일상 사용]
캘린더에서 오늘 할 일 확인
    → 할 일 추가/완료/삭제
    → 매트릭스 뷰로 전환해서 우선순위 확인
    → 드래그로 사분면 재배치
    ↓
[주간 리포트 (자동 생성)]
    → 사분면별 비율 차트
    → 시간 사용 유형 부여
    → 반복 할 일 달성률
    → 공유 카드 생성
    ↓
[내보내기]
    → .md 파일 다운로드 (옵시디언 연동)
```

## 4. 아이젠하워 매트릭스

### 4-1. 4사분면 구조

```
           긴급함 ←──────────→ 안 긴급함
    ┌─────────────────┬─────────────────┐
    │                 │                 │
 중 │   1사분면        │   2사분면        │
 요 │   🔴 즉시 실행   │   🔵 계획 수립   │
 함 │   DO            │   PLAN          │
    │   마감 임박 보고서 │   운동, 자기개발  │
    │                 │                 │
    ├─────────────────┼─────────────────┤
    │                 │                 │
 안 │   3사분면        │   4사분면        │
 중 │   🟡 위임/최소화  │   ⚫ 제거        │
 요 │   DELEGATE      │   DELETE        │
 함 │   이메일, 잡무    │   SNS, 시간낭비  │
    │                 │                 │
    └─────────────────┴─────────────────┘
```

### 4-2. 배치 방식

- **추가 시 (A방식):** 긴급/중요 선택으로 사분면 자동 배치
- **매트릭스 뷰 (B방식):** 드래그 앤 드롭으로 사분면 간 이동 가능
- 두 방식 병행: 추가할 때 간편하게 선택, 나중에 드래그로 자유롭게 재배치

## 5. 반복 할 일 시스템

### 5-1. 반복 주기

| 주기 | 예시 | 동작 |
|------|------|------|
| 매일 | 운동, 영어 공부, 물 2L | 매일 자동 생성 |
| 매주 | 장보기(토), 주간 리뷰(일) | 지정 요일에 자동 생성 |
| 매월 | 월급 관리(25일), 구독 점검(1일) | 지정 날짜에 자동 생성 |
| 매년 | 건강검진(3월), 연말정산(1월) | 지정 월/일에 자동 생성 |

### 5-2. 동작 규칙

- 한번 등록하면 해당 주기마다 캘린더에 자동 생성
- 체크하면 그날치만 완료, 다음 주기에 다시 나타남
- 반복 할 일은 매트릭스에서도 표시 (보통 2사분면에 위치)
- 반복 아이콘 🔁 으로 일회성과 구분

### 5-3. 반복 달성률 시각화

```
📊 이번 주 반복 할 일

매일 운동      ███████░░░ 5/7일 (71%)
매일 영어      ████░░░░░░ 3/7일 (43%)
매주 장보기    ██████████ 완료 ✅
매주 주간리뷰  ░░░░░░░░░░ 미완료 ❌

전체 반복 달성률: 54%
```

> 반복 할 일은 거의 다 2사분면(중요+안긴급)이다. 이게 꾸준히 달성되는지가 인생을 바꾸는 핵심.

## 6. 시간 사용 유형 (바이럴 킥)

### 6-1. 유형 분류

일주일치 완료된 할 일의 사분면 분포 기반으로 유형 부여:

| 유형 | 조건 | 설명 | 아이젠하워 조언 |
|------|------|------|---------------|
| 소방관 | 1사분면 50%+ | 긴급한 불만 끄느라 바쁜 사람 | "당신의 2사분면을 지키세요. 거기에 인생이 있습니다." |
| 전략가 | 2사분면 30%+ | 중요한 일에 집중하는 이상적인 사람 | "당신은 이미 아이젠하워의 길을 걷고 있습니다." |
| 예스맨 | 3사분면 40%+ | 남의 일만 처리하는 사람 | "NO라고 말하는 법을 배우세요. 당신의 시간은 당신 것입니다." |
| 방랑자 | 4사분면 30%+ | 시간을 흘려보내는 사람 | "계획은 아무것도 아니다. 하지만 계획하기가 전부다." |
| 균형자 | 고른 분포 | 모든 영역을 적절히 관리하는 사람 | "균형을 잡은 당신, 이제 2사분면의 비율을 더 높여보세요." |

## 7. 마크다운 내보내기/가져오기

### 7-1. 내보내기 형식

```markdown
# 2026-04-06 (일)

## 🔴 즉시 실행 (긴급 + 중요)
- [x] 클라이언트 보고서 마감
- [ ] 버그 핫픽스 배포

## 🔵 계획 수립 (중요 + 안긴급)
- [ ] EP06 기획서 초안
- [x] 운동 30분 🔁매일

## 🟡 위임/최소화 (긴급 + 안중요)
- [x] 이메일 정리

## ⚫ 제거 (안긴급 + 안중요)
- [ ] SNS 확인

---
반복 달성률: 운동 5/7 (71%) | 영어 3/7 (43%)
유형: 소방관형 (1사분면 62%)
```

### 7-2. 기능

- 일별/주별/월별 단위로 .md 파일 다운로드
- .md 파일 업로드하면 할 일 가져오기 (v1.5)
- 옵시디언 볼트에 넣으면 바로 노트로 활용
- 체크박스 `- [x]` / `- [ ]` 형식은 옵시디언에서 그대로 인식됨
